Subscripts

The sub tag defines subscript text. Subscript text
appears half a character below the normal line, and is
sometimes rendered in a smaller font.
Subscript text can be used for chemical formulas, like H2O.

Superscripts

The sup tag defines superscript text. Superscript text

sometimes rendered in a smaller font.
Superscript text can be used for footnotes, like WWW
